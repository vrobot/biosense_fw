To process raw data files run: python3 raw_data.py filename.csv SENSITIVITY
where SENSITIVITY is typically 1/8192 = 0.00012207031

To process activity average files run: python3 act_avg.py filename.csv

To process test_mode files run: ./test_mode.sh filename.csv SENSITIVITY
where SENSITIVITY is typically 1/8192 = 0.00012207031
